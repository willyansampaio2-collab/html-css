Principais formatações

Negrito / Destaque

Negrito

Nesta frase temos um termo em negrito usando a tag <b> (não semântica).

Destaque

Nesta frase, tenho um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Itálico

Nesta frase, temos um termo em Itálico utilizando a tag < i > (não semântica).

Ênfase

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark> (semântico).

Texto Grande / Texto Pequeno

Texto Grande

Estamos criando um texto grande usando a tag <big> (a tag big foi removida no HTML5 mas funciona ainda).

Texto pequeno

Estamos criando um texto pequeno com a tag <small>
(já no caso da tag small, ela foi mantida e é semântica).

Texto Deletado / Texto Inserido

Texto Deletado

Podemos marcar um texto deletado usando a tag <del> para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

podemos marcar um texto como inserido utilizando a tag <ins> para dar uma ênfase e indicar que ele foi adicionado depois (semântica).

Obs. Também podemos sublinhar um texto com a tag <u> (não semântica).

Texto Sobrescrito / Texto Subscrito

Sobrescrito

Para inserir coisas do tipo x2 utilizamos a tag <sup>.

Subscrito

Para inserir coisas do tipo H2O utilizamosa tag <sub>.
